# Support for Deucarian Package Installer

Use [GitHub Issues](https://github.com/Deucarian/Package-Installer/issues) for reproducible installer-window, catalog, dependency-ordering, install/update/remove, or package-graph defects; documentation gaps; and narrowly scoped feature requests. Search existing issues before opening a new one.

Include the installer version, Unity editor version and operating system, selected channel, affected package IDs and versions, the smallest reproduction, and the complete editor error with local paths, tokens, and private registry data removed.

`main` is the stable package channel. `develop` is a preview channel and may change before promotion. Please reproduce against the applicable current channel before reporting an older snapshot.

Do not disclose vulnerabilities, registry credentials, private repository URLs, or user data in an issue. Follow [SECURITY.md](SECURITY.md) for private security reports.

Support is provided on a best-effort basis. Package-specific runtime defects belong in the affected package repository; Unity Package Manager defects outside the installer are owned by Unity.
